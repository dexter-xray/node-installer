Продвинутый Bash-скрипт для быстрой, безопасной и оптимизированной установки **Remnanode** на отдельный сервер. Разработан специально для работы в условиях жестких сетевых блокировок и высоких нагрузок.

## ✨ Возможности и фичи

* ⚙️ **Полный фоновый деплой:** Автоматически накатывает пакеты, настраивает зависимости, ставит официальный Docker и Certbot.
* 🔒 **Классический TLS:** Автоматический выпуск SSL-сертификатов от Let's Encrypt.
* 🛡️ **Защита от сканеров портов:** Реализована интеллектуальная защита на уровне `iptables`. Полноценные рабочие сессии и порты (SSH, Служебный порт ноды, VPN) находятся в белом списке. Если цензор или активный сканер делает более 3 запросов на скрытые/закрытые порты за 60 секунд — его IP-адрес автоматически улетает в жесткий бан.
* ⚡ **Ultimate Kernel Tuning (sysctl):** В скрипт интегрирован мощный боевой тюнинг ядра для VPN-серверов:
  * Активация алгоритма контроля заторов **BBR** и планировщика пакетов **FQ**.
  * Огромные сетевые буферы сетевого стека для исключения потерь пакетов на гигабитных скоростях.
  * Защита от SYN-flood атак на аппаратном уровне ядра.
  * Увеличенные лимиты файловых дескрипторов и трекинга соединений (`nf_conntrack`) для стабильной обработки сотен тысяч одновременных запросов.
* 🔄 **Умное фоновое автопродление:** Настраивает Cron-задачу для проверки сертификатов. Контейнер перезагрузится только тогда, когда сертификат реально обновится (раз в 2 месяца), не прерывая сессии пользователей каждую ночь.

## 🚀 Быстрый запуск (Установка в одну строку)

Перед запуском убедитесь, что вы направили ваш **домен (A-запись)** на IP-адрес этого сервера, а порты `80` и желаемые вами порты ноды/VPN открыты.

Просто скопируйте и запустите команду в терминале вашего сервера от имени `root` :

Версия: TLS + BBR + Auto-renew

```bash
curl -fsSL https://raw.githubusercontent.com/dexter-xray/node-installer/main/installer.sh | sudo bash
```

Версия: TLS + BBR-X + Net-Shield + Auto-renew

```bash
curl -fsSL https://raw.githubusercontent.com/dexter-xray/node-installer/main/ultimate.sh | sudo bash
```

Версия: Ultimate + dev Xray-core

```bash
curl -fsSL https://raw.githubusercontent.com/dexter-xray/node-installer/main/ult_dev.sh | sudo bash
```

## 📝 Какие данные запрашивает скрипт?

При старте скрипт в интерактивном режиме попросит ввести:

1. **Домен:** Для привязки SSL-сертификата (например, `example.com`).
2. **Email:** Для регистрации в Let's Encrypt и продления SSL.
3. **Secret Key:** Секретный токен связи, скопированный из вашей панели Remnawave.
4. **Служебный порт ноды:** Порт, по которому панель управляет нодой.
5. **Рабочий порт VPN:** Фактический порт подключения пользователей, который будет прописан в Inbound.
6. **Порт SSH:** Текущий порт вашего сервера (По умолчанию: `22`), чтобы скрипт случайно не заблокировал вам доступ к консоли.

## 🛠️ Полезные команды для администрирования

* **Посмотреть логи ноды в реальном времени:**
  ```bash
  cd /opt/remnanode && docker compose logs -f -t
  ```
* **Перезапустить контейнер ноды:**
  ```bash
  cd /opt/remnanode && docker compose down && docker compose up -d
  ```
* **Проверить, что BBR активен:**
  ```bash
  sysctl net.ipv4.tcp_congestion_control
  ```
* **Проверить текущие правила iptables:**
  ```bash
  iptables -L -n -v
  ```
